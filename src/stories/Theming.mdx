# Theming & Tokens

Tokens include colors (background/surface/primary), spacing (4/8/16/24), radii (4/8/12), typography, and elevation.

```tsx
import { ThemeProvider, useThemeMode } from 'my-design-system';

function Toggle() {
  const { mode, toggle } = useThemeMode();
  return <button onClick={toggle}>Mode: {mode}</button>;
}
```
